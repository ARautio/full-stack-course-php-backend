Objects

Object in general

Object is a structure which combined associated array type of key value structures with data and functions.

This helps if we want to have all the operations and data related to some type of content in place.

Some languages with better to use of objects and other less. PHP supports objects greatly. If you are interest about to know about Object oriented programming Wikipedia has a good article.

Wrap up PHP objects

Wrap up

  • Objects are datatype which combines functions and data to single variable. With this a single variable can handle certain set of operations.
  • New object is create with new ClassName(); and functions and data are used with $variable->function.
