Cookies

Introduction

Cookies are way to store information on users computer. Cookies are used often when to determine user without login information or follow the users movement in the site.

Features

In PHP cookies can be used with following content:


    setcookie("name", "value", time() + (86400 * 30), "/");
    // Third parameter tells how long the cookie is stored
    // in seconds (86400 = 1 day)
    $_COOKIE // Cookie array

Security

Cookies are stored in users computer and can be modified so don't rely them for information and don't save any confidential material.
