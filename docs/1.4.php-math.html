PHP Math

Definition

Calculations in PHP are done with two different data types; integer and floating point numbers (which will be called float).

Integers are whole numbers like -3, 0, 100 while floats can present any numerical value like -35.053, 5.434 or 20.000.

Use cases for integers and floats

Integers are usually used for counting (like amount of characters in string) or amount of items available because they are faster to use and reliable to check equality.

Floating numbers are used when the application is dealing with real life values like volumes, monetary values because there we need a decimal precision.

Using integers and floats

<?php
$integer = 4;
$float = 5.33;
print $integer * $integer; // 16 
print $float * $integer; // 21.32 (This is float now)
print $float * $float; // 28.4089
?>

Using strings for math

PHP isn't very strict about different data types.

<?php
$string_number = "5.5";
$float = 5.33;
print $integer * $string_number; // 29.315
print $integer * "k"; // A non-numeric value encountered
?>

is_float and is_int

<?php
$float = 5.52;
$integer = 4;
print var_dump(is_float($float)); // true
print var_dump(is_int($integer)); // true
print var_dump(is_int("2")); // false
print var_dump(is_numeric("2")); // true
?>

intval

Float can be converted to integer with intval function.

<?php
$float = 5.33;
$float_to_int = intval($float);
print $float_to_int; // 5 (Note that we lost precision)
?>

round, floor, ceil

<?php
$float = 5.52;
print round($float); // 6
print floor($float); // 5 (will always round to lower);
print ceil($float); // 6 (will always round to high);
print round($float*10)/10; // 5.5;
?>

number_format

<?php
$float = 5.52;
print number_format($float,1); // 5.5
print number_format($float,5); // 5.52000
print number_format($float*1000,2,".",","); // 5,520.00
?>

rand

Rand function will generate a pseudo random value. This means that there is a changing seed value (which can be for example time) which will be used to calculate the random value.

<?php
print rand(); // Random number between 0 and maximum number;
print rand(10,20); // Random number between 10 and 20.
?>
