CRUD: Intro

CRUD

CRUD comes from words Create, Read, Update, Delete

It's a basic concept to manage data in applications and helps out understanding what components are needed to create a basic application.

Target

For next two week ours target is the create a blog which has following features:

  • You are able to write a blog text and save it database.
  • You are able to read the blog text and see all the blogtexts.
  • Update already written blog texts
  • Delete blog texts

This will be part of your individual project.

Database

We need id, date, author, title, content

How we do that?

Database


			CREATE TABLE blog_entries (
				id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
				date TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
				author VARCHAR(50) NOT NULL,
				title VARCHAR(50) NOT NULL,
				content TEXT NOT NULL
			)
