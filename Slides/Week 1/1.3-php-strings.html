PHP Strings

Definition

String are series of characters like

The 1st event was so amazing!
  • It can be one more lines
    (and lines can be separated by \n or in HTML with <br/>).
  • It can include any kind of characters like numbers alphabets and marks or even html tags.

Strings in PHP

In PHP strings are defined by putting the content inside two quote marks like this:

$feedback_single = 'The 1st event was so amazing!'; $feedback_double = "The 1st event was so amazing!";

The difference between these two are that string with single quote is read as is while string with double quotes will be executed.

Quote examples with Strings

    <?php
  $event = '2nd';
  print 'The $event event was so amazing!'; 
  // 'The $event event was so amazing!' 
  print "The $event event was so amazing!"; 
  // "The 2nd event was so amazing!"
?>

Similar usage with single quotes

Single quotes can be used like double quotes by adding variable separately in between and using dot in between. 

    <?php
  $event = '3rd';
  print 'The ' . $event . ' event was so amazing!'; 
  // "The 3rd event was so amazing!"
?>

Combining HTML and PHP strings

We can also include HTML in PHP. 

      <?php
  $event = '4th';
  print 'The <b>' . $event . '</b> event was so amazing!'; 
?>

When opening this in a browser it will show: The 4th event was so amazing!

Functions

PHP has a lot of ready made functions to edit and handle strings. Following ones you will most likely need in future exercises.

  • print - to output text
  • strtolower and strtoupper - to lowercase and uppercase text
  • strlen - to get length of the string
  • strpos - to find position of character in string
  • substr - to substring string

Print

This function we have already seen earlier but here is a summary. Note that these are also constructs so they can be used without brackets.

<?php
  $text = 'Text';
  print $text;  // Executes as Text
  print $text . $text; // Executes as TextText
?>

Strtolower and strtoupper

<?php
  $text = 'Text';
  print strtolower($text); // This will execute as text
  print strtoupper($text); // This will execute as TEXT
?>

Strlen

This will calculate the amount of characters in string.

<?php
  $text = 'Text 1';
  print strlen($text); // Executes as 6
?>

Strpos

This will count the first position where it finds the character or another string.

<?php
  $text = 'Text 1';
  print strpos($text,'e'); // Executes as 1
  print strpos($text,'xt'); // Executes as 2
?>

Substr

This will get part of the string.

<?php
  $text = 'Longer Text and a lot of unnecessary stuff';
  print substr($text, 7, 8); // Executes as Text and
?>

Let's wrap up PHP strings

  • String are used almost in every web application so learn them well. Understanding that double quotes execute variables inside the string while single quotes don't, helps a lot in the future.
  • Knowning basic functions like strlen and strpos will help later when we get more the data manipulation.
