Loops and arrays

Loops

Loop is a way to execute block of code over and over again until specific rules are filled. The rules depend on a loop type.

While loop

The most simple loop is a while loop

<?php
$is_this_true = true;
while($is_this_true){
	print "Hey";
}
?>

If you try out this loop, why it doens't work?

While loop

To fix a previous issue we need to have something that changes boolean to false.

<?php
$is_this_true = true;
while($is_this_true){
	print "Hey";
	$is_this_true = false;
}
?>

What will this do?

While loop

To get loop go through multiple times, we need to have a condition that doens't change boolean to false immediately. One way is calculating. 

<?php
$rounds = 0;
while($rounds < 10){
	print "Hey";
	$rounds = $rounds +1 ; //Â This can be shorten $rounds++;
}
?>

And now we have Hey 10 times.

For loop

Instead we can use a for-loop:

			<?php
for ($x = 1; $x <= 10; $x++) {
	print "Hello"
} 
?>

Which will print Hello 10 times.

Using for loop

For loop can also print numbers because it knows how many times it has gone already.

				<?php
for ($x = 1; $x <= 10; $x++) {
	print $x . "Hello"
} 
?>

Try this out and try to think why it prints numbers that way.

For loop syntax


for (init counter; test counter; increment counter) {
	code to be executed;
}

For loop syntax

init counter: Here we initialize a variable with start value.
test counter: This one tests out if we should do one more round. If this is true then we do.
increment counter: This changes the value so one closer to limit of test counter.
