Introduction Databases

Databases means a set of data which is organised certain way so everything from excel sheets to complex set of data organisation can be called as databases. In this course and mostly in web development, a relational database is used.

If to simplify relational databases are a bit like an excel file. They can contain multiple tables and each table can consist from rows and columns.
