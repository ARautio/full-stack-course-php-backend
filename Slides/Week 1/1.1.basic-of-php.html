What is needed for PHP?

  1. PHP is a server-side scripting language which means that the page needs to be generated everytime user enters the page.
  2. Unlike HTML or Javascript, it cannot be done on browser so we need an application to read and convert the code to readable text so that browser can show it.

How PHP works

Running PHP locally

  • Serving webpages online we would need a server or webhosting to ensure that pages are available all the time.
  • But for development we can install same applications to our own computer and run PHP files through it.
  • For this purpose we are using MAMP.

Installing and running MAMP

  1. Go to address https://www.mamp.info/en/ and download Windows version from MAMP.
  2. Double click installation package and install application to your folder by following instructions.
  3. Open MAMP application and click XXXX.

First PHP file

  1. Go to folder C:/MAMP/htdocs/ (or where you did install MAMP)
  2. Create a file called feedback.php
  3. Add following content to file:
    <?php print "Hello students"; ?>
  4. Open Browser and go to http://localhost/
