Arrays

Arrays

Array is a datatype which includes a set of items of other datatype.

For example an array can be "Yellow", "Blue", "Red, "Purple".

Arrays in PHP

Arrays in PHP contains three different types of set of items (indexed, associated and multidimensional). First and the easiest to understand is indexed array. This means that each of the items in array has a number which is refers.

For example in our previous set of colours, "Yellow" would be 0 and "Red" would be 2.

Indexed array in PHP

				<?php
$colors = array("Yellow","Blue","Green","Red","Purple");
print $colors[0]; // Prints Yellow
print $colors[2]; // Prints Red
?>

Indexed array in PHP

<?php
$colors = array("Yellow","Blue","Green","Red","Purple");
$colors[0] = "White";
print $colors[0]; // Prints White
?>

Indexed array in PHP and loops

<?php
$colors = array("Yellow","Blue","Green","Red","Purple");
for($x = 0; $x < sizeof($colors); $x++){
	print $colors[$x]; // Prints colors one by one
}
?>

Associated array

Difference between indexed array and associated arrays is that the keys can be string instead of integers from 0.

From example ("top" => "yellow", "middle" => "green", "bottom" => "red")

Associated array in PHP

				<?php
$colors = array("top" => "yellow", "middle" => "green", "bottom" => "red");
print $colors["top"]; // Prints yello
print $colors["bottom"]; // Prints red
?>

Associated array in PHP and foreach

<?php
$colors = array("top" => "yellow", "middle" => "green", "bottom" => "red");
foreach($colors as $color){
	print $color;
}
?>
